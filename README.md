# 대구프론티어센터 정책표 (직원용)

- UI 컨셉: 박스형/표형(상품 안내 페이지 느낌) + 시인성 강화
- 정책 데이터: `data/유선.xlsx` 기준으로 `js/policy-data.js`를 생성한 버전입니다.

## 사용 방법
1. 이 폴더를 GitHub에 업로드
2. GitHub Pages 활성화 (Settings → Pages)
3. `index.html` 접속

## 정책 수정
- 원본 엑셀: `data/유선.xlsx`
- **엑셀 수정 후에는** `js/policy-data.js`를 다시 생성해야 합니다.
  - 현재는 제가 생성해 둔 상태이며, 필요하면 자동생성 스크립트도 추가해드릴 수 있습니다.
